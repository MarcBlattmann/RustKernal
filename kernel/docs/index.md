# Pursuit OS Documentation

Welcome to Pursuit OS! This is a simple operating system 
written entirely in Rust.

## Quick Start

1. Use the Start Menu to launch apps
2. Create files with the File Explorer
3. Write code with the Code Editor
4. Use the Terminal for commands

## Available Applications

### Code Editor
A syntax-highlighting text editor for writing:
- Rust (.rs) files
- Pursuit Apps (.pa) files
- Plain text files

Features:
- Line numbers
- Syntax highlighting
- Save with Ctrl+S

### File Explorer
Browse and manage your files:
- View files and directories
- Create new files/folders
- Delete files
- Open files in editor

### Terminal
Command-line interface with commands:
- ls - List files
- cat - View file contents
- touch - Create file
- rm - Delete file
- mkdir - Create directory
- run - Launch apps
- help - Show all commands

### Documentation
You're reading it! Browse the docs
to learn about PursuitScript and
creating your own applications.

## Tips

- Drag windows by their title bar
- Click X to close a window
- Use keyboard shortcuts for speed
